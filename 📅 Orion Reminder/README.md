# Orion Reminder

Ein modernes, dunkles Expo-React-Native Starter-Projekt für eine Premium-Erinnerungs- und Produktivitäts-App.

## Features
- Dashboard mit Tageszielen
- Kalenderansicht
- Aufgaben und Erinnerungen
- Statistiken und Fortschrittsübersicht
- Glossy Glassmorphism UI
- Dark Mode

## Starten

1. `npm install`
2. `npm start`
3. `npm run android` oder `npm run ios`

## Struktur
- `App.tsx`: Einstiegspunkt
- `src/navigation`: Bottom-Tab-Navigation
- `src/screens`: App-Bildschirme
- `src/components`: wiederverwendbare UI-Komponenten
- `src/theme.ts`: Farbpalette und Stile
